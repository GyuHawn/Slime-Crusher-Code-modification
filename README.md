# 슬라임크러셔

## 개발인원 및 기간
2024.01.15 ~ 2024.02.25 (1인 개발)

## 게임소개
Unity 2D로 개발한 로그라이크 형식의 모바일 게임입니다.
스테이지별로 슬라임을 쓰러트려 마지막 스테이지까지 클리어를 목표로 합니다.
