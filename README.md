## 게임소개
슬라임 크러셔는 유니티 2D를 활용하여 개발한 모바일 로그라이크 게임입니다.
터치와 슬라이드를 이용하여 몬스터를 사냥하는 방식이고, 여러 아이템과 다양한 슬라임을 추가하여 매 게임마다 새로운 방식 경험을 하도록 개발하였습니다.

## 사용기술
### [Unity]
### [C#]

## 에러사항
* 몬스터와 보스마다 각각의 공격과 스킬을 추가
* 부족한 리소스로 인한 아쉬운 디자인 개선 필요
